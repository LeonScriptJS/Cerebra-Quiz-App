# Cerebra Quiz App

Cerebra is a web-based IQ quiz application designed to provide users with a clean and engaging way to test their reasoning ability through a series of interactive questions.

Many online IQ tests are cluttered with advertisements, hidden behind paywalls, or poorly designed. Cerebra was built to offer a **simple, distraction-free experience** where users can focus entirely on solving problems and enjoying the challenge.

The application presents a sequence of multiple-choice questions, provides immediate visual feedback when answers are selected, and concludes with a results page that estimates the user's IQ range based on their performance.

Beyond the user experience, this project also demonstrates the ability to build a **structured frontend application using React**, reusable components, responsive layouts, and interactive UI behavior.

# Installation

To run this project locally:

### 1. Clone the repository

```bash
git clone https://github.com/LeonScriptJS/Cerebra-Quiz-App.git
```

### 2. Navigate into the project

```bash
cd Cerebra-Quiz-App
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

The application will run locally on:

```
http://localhost:5173
```

# How It Works

1. The user lands on the homepage and starts the quiz.
2. Each question is presented with multiple choice answers.
3. When a user selects an option, the interface provides **visual feedback** indicating whether the answer is correct or incorrect.
4. After completing the quiz, the user is redirected to a **results page** showing their estimated IQ range and performance feedback.
